# Moho Script Helper

Lua autocomplete and stubs for Moho 14.3+ scripting.

## Features

- 🚀 **Autocomplete** for Moho API (14.3+)
- 📁 **Moho stubs** file (`moho-stubs.lua`)
- ⚙️ **Automatic Lua Language Server** configuration
- 🎯 **Smart suggestions** for `moho.` and `LM.` APIs

## Requirements

- VS Code 1.125.0 or higher
- [Lua Language Server](https://marketplace.visualstudio.com/items?itemName=sumneko.lua) extension

## Supported Moho Versions

| Version | Support |
|---------|---------|
| Moho 14.3 | ✅ Full support |
| Moho 14.x | ✅ Full support |
| Moho 13.x | ⚠️ Limited support |

## Installation

1. Install from the VS Code Marketplace
2. Restart VS Code
3. Open a `.lua` file
4. Start typing `moho.` – autocomplete will appear!
